Depth-first traversal generator¶

A depth-first traversal generator generates events for a depth-first traversal, which always extends a path as far as possible before backtracking.

Events generated¶

Example¶

>>> import yaupon
>>> from yaupon.traversal import *
>>> g = yaupon.Graph(edges = [(0,1),(1,2),(2,3),(0,4),(4,5),(5,3),(0,6),(6,7)])
>>> generator = depth_first_generator(g)
>>> generator.bootstrap(0)
True
>>> for event in generator.events():
...     print(event)
...
